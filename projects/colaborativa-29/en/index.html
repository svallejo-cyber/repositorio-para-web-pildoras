Collaborative pill · Isaval team

From daily checks to preventive color criteria
Useful history of Dromont checks and calibrations to anticipate deviations before production

Pedro José Expósito Navarro · Twenty-ninth collaborative contribution to the hub on quality control, Dromont calibration and historical color traceability

The problem

Dromont already performs daily quality checks on critical colorants, but the operational data is used almost only to decide whether a valve is OK or needs recalibration at that moment.

The current spreadsheet overwrites the previous week. That supports daily execution, but prevents building historical memory and learning which colorants drift more, which ones stabilize after recalibration and where intervention should happen before production.

Today the deviation is detected. The proposal is to start understanding its behavior.

What already works

Operators already use an Excel file connected to the Dromont scale that captures weights automatically and compares them with tolerance thresholds defined for each colorant.

This simplifies the work, avoids transcription errors and keeps the control routine light: if the result is within range, the process moves on; if not, the valve is recalibrated and checked again.

Real example of the table used for pre-production checks and recalibration.

What the pill proposes

Pedro proposes to stop treating that Excel file as an ephemeral tool and turn it into a historical source of operational knowledge.

The first step would be to preserve and collect checks and recalibrations automatically in order to build useful statistics. The next step would be to cross that data with ERP and maintenance records to relate deviations to pump changes, valves, batches or mechanical interventions.

What decisions it would enable

Less unnecessary control

Identify low-incidence valves and review whether check frequency can be adjusted.

Selective follow-up

Separate colorants that recover well after calibration from those requiring closer monitoring.

Better tuned tolerances

Refine limits using real history instead of only the original theoretical settings.

Early alerts

Detect valves that remain out of range after calibration and escalate the issue faster.

Business impact

Moving from daily control to preventive criteria would reduce late interventions, improve pre-production preparation and help decide earlier when to adjust, monitor, recalibrate or replace a component.

It would also open the door to a historical behavior dashboard per colorant, with technical and predictive value for the plant.
