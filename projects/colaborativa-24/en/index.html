From theoretical limits to real acceptance ranges

Sergio Huertas · Twenty-fourth collaborative contribution to the hub on quality control, specification review and live use of production history to adjust acceptance ranges

The problem

Quality control has long relied on defined specifications for each product, but there had never been a systematic review of acceptance ranges based on the actual reality of production history.

That meant working with theoretical limits without a tool able to show whether those limits still reflect real manufacturing behavior, or whether formulation changes and fewer adjustments would improve production flow.

The opportunity was not to generate more data, but to turn five years of production history into technical judgement for reviewing specifications on a real basis.

What has been built

Sergio developed a Python application that consolidates quality-control history from Excel and CSV files, cleans and normalizes production data and analyzes each combination of product, color and property.

On top of that, the tool proposes specification ranges using median ± 20%, compares the current specification against the proposed one and generates technical recommendations and exportable reports.

What it does operationally

Ingestion

It consolidates historical files and avoids reprocessing already-treated sources through incremental traceability.

Analytics

It evaluates density, viscosity, tinting strength and pH by reference, color and property combination.

Recommendation

It proposes new ranges and compares the current specification against the suggested one through parameterized rules.

Live use

The program is already running and being used in the quality area, with a web interface and report export.

Relevant findings from the project log

  • The flow is implemented end to end: ingestion, cleaning, analytics, recommendations, UI and exports.
  • The observed state includes 6,179 consolidated rows and a repeated signal of 1,584 viscosity values outside the configured range.
  • The tool already opens the door to a stricter review of whether current ranges reflect actual production reality or whether recentering or reformulation is needed.
Collaborative conclusion: this proposal turns quality-control history into a working tool that already helps review specifications on a real basis, detect structural deviations and reduce unnecessary manufacturing adjustments.
